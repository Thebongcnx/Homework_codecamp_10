    2.Grid และ flex แตกต่างกันอยางไร
    • Grid:ใช้เมื่อ items ด้านในเรามีการจัดเรียงตัวที่ไม่ได้มีแค่แถวเดียว แต่มีลักษณะเป็น column กับ row (2D)
    • Flex:ใช้เมื่อ layout มีลักษณะการวางของต่างๆเป็นแถวเดียว ไม่แนวตั้งก็นอน (1D)
    3.Justify และ align แตกต่างกันอยางไร
    • Justify คือ การกระจายข้อความให้เต็มบรรทัด
    • Align คือ การจัดตำแหน่งให้ข้อความไปทางใดทางหนึ่ง
