Review CSS #002

1.property และ value คือ
  • property คือ คุณสมบัติในการจัดรูปแบบการแสดงผล เช่น color สำหรับกำหนดสี, font-size สำหรับกำหนดขนาดตัวอักษร
  • value เป็น ค่า ที่เรากำหนดให้กับ property ต่างๆ เช่น color:white, font-size:14px
2.property จาก Precamp
  • font-family (รูปแบบตัวอักษร)
  • color (สีของวัตถุ)
  • background-color (สีของพื้นหลัง)
  • margin (พื้นที่ภายนอกวัตถุ)
  • padding (พื้นที่ภายในกรอบวัตถุ)
  • border (เส้นรอบวัตถุ)
3.value unit มี 2 ประเภทหลักๆ
  1. Relative Length : แบบกำหนดอัตราส่วน
    • px (pixel)
    • em (emphasize)
    • ex (x-height)
    • % (percent)
  2. Absolute Length : อัตราส่วนตายตัว
    • in (inches)
    • cm (centimeters)
    • mm (millimeters)
    • pt (points)
    • pc (picas)
