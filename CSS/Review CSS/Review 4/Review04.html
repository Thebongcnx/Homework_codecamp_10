1.Position คืออะไร และมีอะไรบ้าง
  • Position คือ property หนึ่งของ css
  • มีทั้งหมด 7 แบบ
    1. static
    2. relative
    3. fixed
    4. absolute
    5. sticky
    6. initial
    7. inherit
2.Z-index คืออะไร และใช้อย่างไร
  • z-index คือตัวกำหนดจัดการ การซ้อนทับของ layout ในแกน z
  • การใช้ z-index นั้น ต้องมีการกำหนด position ให้ทุกครั้ง
